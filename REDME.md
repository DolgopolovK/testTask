Порядок команд для запуска
1) mvn clean package

2) java -jar -Xmx7m target/airports-search-*.jar [номер колонки в файле для поиска. Например 2]


Тестовое задание стажировки Java-разработчиков

Библиотека autocomplete вводимого текста

Требуется написать консольное Java-приложение (JDK 11), позволяющее быстро искать
данные аэропортов по вводимому пользователем тексту.

Данные для программы берутся из файла . В нем находится таблица аэропортов
со свойствами в формате CSV. За что отвечает каждая колонка – не важно.

Сборка проекта осуществляется с помощью Maven. После сборки исходного кода командой
, получаем в качестве артефакта для запуска.
